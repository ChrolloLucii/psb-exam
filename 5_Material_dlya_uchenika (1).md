# Markdown - всё гениальное просто

> Привет! Это материал по Markdown.

---

## Что это вообще такое 🤔

Представь, что ты написал конспект. Получилась серая стена из букв - скучно, и сложно перечитывать. А теперь добавь пару символов - и текст превратится в красивую страницу с заголовками, списками и выделениями.

**Markdown - это способ оформлять текст обычными символами с клавиатуры.** Ты печатаешь звёздочки, решётки, чёрточки - а программа читает их и делает красиво.

Принцип всего Markdown один:

> **Оборачиваешь текст символами - он меняет вид.**

## Где ты это уже встречал

Скорее всего, ты пользовался Markdown и не знал об этом:

- **Discord** - когда пишешь `**текст**`, чтобы выделить жирным в чате.
- **GitHub** - там почти каждый проект начинается с файла `README.md` (это Markdown).
- **Reddit, Telegram, Notion, WhatsApp** - везде он, просто работает незаметно.

Формату больше 20 лет (придумали в 2004-м), и он не устарел - именно потому что он простой.

## Весь синтаксис - по порядку

Дальше для каждого приёма: **что ты пишешь** и **что видишь**. Лучший способ понять - открыть редактор (ссылки в конце) и повторить.

### 1. Заголовки

```
# Большой заголовок
## Поменьше
### Ещё мельче
```

Решётка `#` делает заголовок. Одна - самый крупный, две - поменьше, три - ещё мельче. Чем больше решёток, тем мельче текст.

**Главное правило:** после решётки обязательно **пробел**. `#Заголовок` не сработает, а `# Заголовок` - да.

### 2. Жирный, курсив, зачёркнутый

```
**жирный**          → жирный
*курсив*            → курсив
~~зачёркнутый~~     → зачёркнутый
***жирный курсив*** → сразу и то, и то
```

### 3. Списки

```
Маркированный:        Нумерованный:
- молоко              1. зайти на сервер
- хлеб                2. построить дом
- кот                 3. выжить ночью
```

Маркированный - **дефис и пробел** в начале строки. Нумерованный - **цифра, точка, пробел**.

Фишка: в нумерованном можешь хоть везде писать `1.` - Markdown сам посчитает по порядку.

### 4. Ссылки и картинки

```
[мой канал](https://youtube.com)
![кот](cat.png)
```

Ссылка: в **квадратных** скобках - видимый текст, в **круглых** - куда ведёт. Картинка - то же самое, но впереди ставишь **восклицательный знак** `!`.

### 5. Код

Внутри строки:  Нажми `Ctrl + C`, чтобы скопировать.

Целым блоком:
```
print("Привет, мир!")
```

*(три обратных апострофа ``` подряд, без пробелов)*

Одинарными обратными кавычками `` ` `` оборачиваешь код **внутри предложения** (команду, название клавиши, путь к файлу). Тремя - **целый блок кода**, он встанет в рамочку моноширинным шрифтом.


### 6. Таблицы

```
| Игра      | Оценка |
|-----------|--------|
| Майнкрафт | 10/10  |
| Роблокс   | 5/10   |
```

Колонки разделяешь палочкой `|`. **Вторая строка из дефисов** - это «шапка», она отделяет заголовки от данных. Не обязательно ровнять палочки идеально - Markdown сам выровняет.

### 7. Цитаты

```
> Это цитата.
> Удобно для важных мыслей.
```

Знак «больше» `>` в начале строки делает цитату - текст с полоской слева. Так оформляют важные мысли, правила, чужие слова.

## Попробуй прямо сейчас

Собери на Markdown свою страничку-визитку. Должно получиться примерно так:

```
# Мой профиль

Я люблю **...** и *обожаю* ... .

- люблю ... 
- ненавижу ...

> Мой девиз: .
```

Открой редактор, набери это - и увидишь, как символы превращаются в красивую страницу. Потом добавь своё: ссылку на свой канал, таблицу с любимыми играми, картинку.

Поздравляю - теперь ты умеешь читать и писать то, на чём держится вся современная документация!.
